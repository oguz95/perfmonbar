Perfmonbar

Displays performance counters on the Windows Taskbar

Download installer View on GitHub

Displays performance counters on the Windows Taskbar. This is a replacement for CureInfo. It has support for xml configuration and is compatible with Vista/7 (Taskbar Translucency, etc...)

Original PerfmonBar Page

Supported Operating Systems

Perfmonbar runs on Windows XP (SP3, SP2 might or might not work), Vista, 7, 8 and 8.1 both 32-bit and 64-bit
